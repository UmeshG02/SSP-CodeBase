35. Scope of Variables
The LEGB rule in Python describes the search order for variable names when
they are referenced in the code. LEGB stands for:
1. L: Local
2. E: Enclosing Function Locals
3. G: Global
4. B: Built-in
Local scope: Refers to variables defined within the current function or block.
Enclosing scope: Refers to variables in any enclosing functions (functions that
contain the current function).
Global scope: Refers to variables defined at the top level of the module or
script.
Built-in scope: Refers to Python's standard library functions, exceptions, and
built-in objects (e.g., print(), len()).
